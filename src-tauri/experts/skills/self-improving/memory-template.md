# Memory Template Notice

Do not create a self-improving memory file from this bundle. iyw-claw already
owns durable memory under `~/.iyw-claw/` and exposes policy-aware host tools.

For operational reflection state, initialize only:

```text
~/.iyw-claw/self-improving/heartbeat-state.md
~/.iyw-claw/self-improving/reflections.md
```

Use [setup.md](setup.md) for their minimal content.
